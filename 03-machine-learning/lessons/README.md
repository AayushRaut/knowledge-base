## Suggested order

Work top to bottom: build the foundations and the scikit-learn workflow, then
supervised methods, then unsupervised methods, and finish with the evaluation and
generalization tools that keep your models honest. Reinforce each area with the
[exercises](../exercises/README.md), then apply everything in the
[projects](../projects/README.md).
