# Module 3 — Machine Learning

> Classical machine learning end-to-end — the paradigms, algorithms, and the
> scikit-learn workflow, taught **intuition → theory → scikit-learn**. This is
> Module 3 of the [Master Curriculum](../.claude/MASTER_CURRICULUM.md).

**Status:** 🟢 Complete · **Difficulty:** beginner → advanced
**Estimated time:** ~40–50 hours · **Last reviewed:** 2026-07-02

---

## 🎯 Learning outcomes

By the end of this module you will be able to:

- Explain the ML paradigms (supervised, unsupervised, semi-supervised, reinforcement) and pick the right one.
- Build leak-free scikit-learn pipelines for regression and classification.
- Apply clustering and dimensionality reduction to unlabeled data.
- Choose and interpret the right evaluation metrics, and cross-validate honestly.
- Diagnose bias/variance, over- and under-fitting, and act on it.

## 📋 Prerequisites

[Module 1 — Python for AI Engineering](../01-python-languages/README.md) (NumPy,
Pandas, feature engineering) and [Module 2 — Mathematics for AI](../02-mathematics-foundations/README.md)
