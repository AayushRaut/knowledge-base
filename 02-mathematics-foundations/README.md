# Module 2 — Mathematics for AI

> **Only the mathematics you need for AI** — linear algebra, probability &
> statistics, calculus & optimization, and information theory. Every topic is
> taught **intuition → mathematics → Python**. This is Module 2 of the
> [Master Curriculum](../.claude/MASTER_CURRICULUM.md).

**Status:** 🟢 Complete · **Difficulty:** beginner → intermediate
**Estimated time:** ~30–40 hours · **Last reviewed:** 2026-07-02

---

## 🎯 Learning outcomes

By the end of this module you will be able to:

- Work with vectors and matrices as both geometry and data, and read the linear
  algebra behind ML models.
- Use eigen-decomposition and understand PCA.
- Reason about uncertainty with probability, statistics, and Bayes' rule.
- Compute gradients and explain how gradient descent and optimizers train models.
- Explain entropy, cross-entropy, and KL divergence — and why cross-entropy is
  the standard classification loss.

## 📋 Prerequisites

Basic Python and [Module 1 — Python for AI Engineering](../01-python-languages/README.md),
especially [NumPy](../01-python-languages/lessons/numpy.md).
