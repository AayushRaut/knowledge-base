## Suggested order

Work top to bottom: linear algebra gives you the language of data and models,
probability/statistics the language of uncertainty, calculus/optimization the
machinery of learning, and information theory the loss functions that tie it all
together. Reinforce each area with the [exercises](../exercises/README.md), then
apply everything in the [projects](../projects/README.md).
