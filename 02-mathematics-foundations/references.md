# Module 2 References

> Curated, verifiable resources for **[Mathematics for AI](README.md)**. Only
> canonical sources are listed.

---

## 📘 Books

- **Mathematics for Machine Learning** — Marc Peter Deisenroth, A. Aldo Faisal,
  Cheng Soon Ong. The single best free reference for this module — linear
  algebra, calculus, probability, and optimization aimed squarely at ML. Free
  PDF: <https://mml-book.github.io/>.
- **Deep Learning** — Ian Goodfellow, Yoshua Bengio, Aaron Courville. Part I
  covers the applied math foundations. Free: <https://www.deeplearningbook.org/>.
- **Think Stats** (2nd ed.) — Allen B. Downey. Practical, code-first statistics.
  Free: <https://greenteapress.com/wp/think-stats-2e/>.
- **Introduction to Linear Algebra** — Gilbert Strang. A classic linear algebra text.

## 📄 Official Documentation

- [NumPy — linear algebra](https://numpy.org/doc/stable/reference/routines.linalg.html)
- [NumPy — random generator](https://numpy.org/doc/stable/reference/random/generator.html)
- [SciPy — stats](https://docs.scipy.org/doc/scipy/reference/stats.html) ·
  [SciPy — optimize](https://docs.scipy.org/doc/scipy/reference/optimize.html)
- [SymPy](https://docs.sympy.org/latest/) (symbolic differentiation)

## ▶️ Video Courses

- **3Blue1Brown** — *Essence of Linear Algebra* and *Essence of Calculus* (the
  best visual intuition available): <https://www.youtube.com/@3blue1brown>.
- **StatQuest with Josh Starmer** — probability and statistics explained simply:
  <https://www.youtube.com/@statquest>.
- **Khan Academy** — linear algebra, calculus, probability, and statistics:
  <https://www.khanacademy.org/>.

## 🌐 Interactive

- [Seeing Theory](https://seeing-theory.brown.edu/) — visual introduction to
  probability and statistics (Brown University).
