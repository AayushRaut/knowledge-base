## Suggested order

Work top to bottom: master raw text handling first, then how text becomes
vectors, then the linguistic structure tools, then the tasks themselves, and
finish with the metrics that tell you whether any of it worked. Reinforce each
group with the [exercises](../exercises/README.md), then apply everything in the
[projects](../projects/README.md).
