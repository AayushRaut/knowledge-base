# Module 5 — Natural Language Processing

> Complete NLP — from raw-text preprocessing and tokenization through embeddings
> and linguistic analysis to classification, sentiment, seq2seq tasks, and how to
> evaluate them. Taught **intuition → concepts → Python**. This is Module 5 of
> the [Master Curriculum](../.claude/MASTER_CURRICULUM.md).

**Status:** 🟢 Complete · **Difficulty:** beginner → intermediate
**Estimated time:** ~35–45 hours · **Last reviewed:** 2026-07-02

---

## 🎯 Learning outcomes

By the end of this module you will be able to:

- Clean, normalize, and tokenize text — including subword tokenization (BPE/WordPiece).
- Represent text as vectors: BoW/TF-IDF, Word2Vec/GloVe/FastText, and sentence embeddings.
- Extract structure with POS tagging, NER, and dependency parsing (spaCy).
- Build and honestly evaluate text classifiers and sentiment systems.
- Explain summarization, translation, and QA — and score them with BLEU/ROUGE/F1/perplexity.
