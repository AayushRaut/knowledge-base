## Suggested order

Work top to bottom: foundations give you the building blocks, training the
machinery that makes them learn, then the architectures that exploit structure in
images and sequences, the generative models that create data, and finally deep
fluency in the frameworks you'll use daily. Reinforce each group with the
[exercises](../exercises/README.md), then apply everything in the
[projects](../projects/README.md).
