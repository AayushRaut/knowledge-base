# Module 4 — Deep Learning

> Neural networks from the perceptron to generative models — architectures,
> training machinery, and the PyTorch/TensorFlow toolchain, taught
> **intuition → mathematics → PyTorch**. This is Module 4 of the
> [Master Curriculum](../.claude/MASTER_CURRICULUM.md).

**Status:** 🟢 Complete · **Difficulty:** beginner → advanced
**Estimated time:** ~45–60 hours · **Last reviewed:** 2026-07-02

---

## 🎯 Learning outcomes

By the end of this module you will be able to:

- Explain and implement neural networks: perceptrons, MLPs, activations, losses.
- Derive and verify backpropagation, and choose/configure optimizers.
- Train reliably with regularization, normalization, and a correct training loop.
- Build CNNs for images and RNN/LSTM/GRU models for sequences.
- Explain autoencoders and GANs, and implement both in PyTorch.
- Work fluently in PyTorch and navigate TensorFlow/Keras.
