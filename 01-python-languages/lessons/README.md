## Suggested order

Work top to bottom. Lessons 1–8 build language mastery, 9–14 the data stack, and
15–20 the engineering practices you need to ship production AI code. Reinforce
each group with the [exercises](../exercises/README.md), then apply everything in
the [projects](../projects/README.md).
