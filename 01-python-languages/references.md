# Module 1 References

> Curated, verifiable resources for **[Python for AI Engineering](README.md)**.
> Only canonical sources are listed. When a deep link is uncertain, the official
> documentation root is given.

---

## 📘 Books

- **Fluent Python** (2nd ed.) — Luciano Ramalho. Deep coverage of the data model,
  iterators, decorators, and idiomatic Python.
- **Effective Python** (2nd ed.) — Brett Slatkin. 90 focused, practical best
  practices.
- **Python for Data Analysis** (3rd ed.) — Wes McKinney (author of pandas). NumPy
  and pandas for real data work.
- **Architecture Patterns with Python** — Harry Percival & Bob Gregory. Clean
  architecture, testing, and design patterns in Python.
