# Module 1 — Python for AI Engineering

> Python **specifically for AI engineering**: the language mastery, numeric/data
> stack, and production practices you need before touching ML. This is Module 1
> of the [Master Curriculum](../.claude/MASTER_CURRICULUM.md#module-1--python-for-ai-engineering).

**Status:** 🟢 Complete · **Difficulty:** beginner → intermediate
**Estimated time:** ~40–55 hours · **Last reviewed:** 2026-07-02

---

## 🎯 Learning outcomes

By the end of this module you will be able to:

- Write idiomatic, typed, testable Python using its data model and advanced features.
- Use NumPy, Pandas, SciPy, and Matplotlib/Seaborn for real data work.
- Clean, wrangle, and engineer features without data leakage.
- Manage environments, package code, and structure a production-grade project.
- Apply logging, configuration, concurrency, and testing the way AI teams do.

## 📋 Prerequisites

Basic Python, SQL, Git, and Linux commands (per the
[curriculum prerequisites](../.claude/MASTER_CURRICULUM.md#3-prerequisites)).

## 🗺️ Suggested path

1. Work the **lessons** top to bottom (core → advanced → data stack → tooling → production).

3. Consolidate with the two **[assignments](assignments/README.md)** and the **[mini projects](projects/README.md)**.
