## Suggested order

Work top to bottom: the four attention lessons build the core mechanism piece by
piece, the architecture lessons assemble it into the full model (and then you
build one yourself), and the variants lessons map the design space that every
modern model — BERT, GPT, T5, ViT — lives in. Reinforce each group with the
[exercises](../exercises/README.md), then apply everything in the
[projects](../projects/README.md).
